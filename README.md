# App Lista de Tarefas

## Descrição

O App Lista de Tarefas é um aplicativo web desenvolvido em PHP, utilizando um banco de dados MySQL, que permite aos usuários criar, alterar e excluir tarefas. Com este aplicativo, os usuários podem gerenciar suas tarefas de forma eficiente, adicionando novas tarefas, atualizando informações existentes e excluindo tarefas concluídas ou desnecessárias.

## Funcionalidades

O App Lista de Tarefas possui as seguintes funcionalidades:

1. **Incluir Tarefa**: Os usuários podem adicionar novas tarefas à lista. Cada tarefa possui um título, uma descrição opcional e uma data de conclusão.

2. **Alterar Tarefa**: Os usuários podem editar as informações de uma tarefa existente, como título, descrição e data de conclusão.

3. **Excluir Tarefa**: Os usuários podem remover uma tarefa da lista quando ela estiver concluída ou não for mais necessária.
